## Microcontroller connection

| **ePaper Display** | **Microcontroller** |
| ------------------ | ------------------- |
| VCC                | 3.3V                |
| GND                | GND                 |
| DIN (MOSI)         | GPIO23              |
| MISO (MISO)        | GPIO19              |
| CLK (SCK)          | GPIO18              |
| CS                 | GPIO5               |
| DC                 | GPIO22              |
| RST                | GPIO21              |
| BUSY               | GPIO4               |
